Latar Belakang Proyek

Arsitektur pusat data tradisional menghadapi tantangan serius dalam memenuhi kebutuhan komputasi modern, seperti elastisitas sumber daya, ketersediaan layanan, dan keamanan sistem. Pengelolaan infrastruktur fisik secara mandiri meningkatkan kompleksitas operasional dan biaya investasi.

Melalui pendekatan cloud computing berbasis Platform as a Service (PaaS), aplikasi ini menunjukkan bagaimana pusat data modern menyederhanakan pengelolaan infrastruktur dengan mengabstraksi perangkat keras dan menyediakan lingkungan aplikasi yang siap pakai, aman, dan dapat diskalakan.

Aspek Keamanan Cloud

  • Enkripsi komunikasi menggunakan protokol HTTPS dan SSL.
  • Isolasi aplikasi dalam lingkungan cloud terkelola.
  • Eliminasi risiko server-side attack karena tidak adanya backend.
  • Penerapan konsep shared responsibility antara cloud provider dan pengguna.
