Latar Belakang dan Permasalahan

Pusat data konvensional memiliki keterbatasan dalam hal skalabilitas, ketersediaan layanan, dan efisiensi pengelolaan sumber daya. Pengelolaan infrastruktur fisik memerlukan biaya tinggi serta kompleksitas operasional yang besar.

Cloud computing menawarkan solusi melalui penyediaan sumber daya komputasi berbasis layanan. Proyek ini mensimulasikan penerapan konsep tersebut melalui web application berbasis Platform as a Service (PaaS) yang merepresentasikan desain pusat data cloud modern.

Alur Arsitektur Sistem Cloud

Alur arsitektur berikut menggambarkan proses kerja sistem secara menyeluruh, mulai dari pengguna hingga proses analisis dan visualisasi data pada lingkungan cloud.

User

Pengguna mengakses aplikasi melalui web browser.

Internet

Media komunikasi antara pengguna dan cloud.

Keamanan (HTTPS / SSL)

Enkripsi komunikasi data untuk menjaga keamanan.

Cloud Platform (PaaS)

Aplikasi di-host pada GitHub Pages sebagai layanan Platform as a Service.

Data Processing System

Pemrosesan data dilakukan di sisi klien menggunakan JavaScript.

Web Application & Visualization

Hasil analisis ditampilkan dalam bentuk dashboard analitik.
